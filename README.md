# Live News Aggregator (Advanced)
This project scrapes multiple news sites and provides a beautiful Streamlit frontend with advanced features:
- Card-style headlines with logos and category badges
- Dark/Light mode toggle
- Breaking news ticker
- Word cloud of trending keywords
- Source statistics chart
- Search, filters, auto-refresh
- Favorites and keyword alerts

Run locally:
```bash
pip install -r requirements.txt
streamlit run app.py
```
